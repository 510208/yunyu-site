事前準備 ​

安裝 Python 執行環境 ​

就如先前所述，這個機器人採用 Python 搭建。這代表了如果你也想要使用它，你也必須要安裝 Python 環境。因此請跟著說明的腳步，安裝適合你系統的 Python 執行時吧！

下載 Python ​

對於Windows(這裡使用的是Windows 10)，預設並沒有內建Python環境，因此我們會需要前往下載。

前往下載Python 3.10

WARNING

不要安裝微軟商店提供的 Python，那會造成程式無法順利運作！

上面是Python官網提供的下載選項，如果你的電腦跟我一樣是Win10，請選擇 Windows installer (64-bit) 選項，這是給64位元的作業系統的，如果你不知道你是不是使用64Bit作業系統，建議你安裝 Windows installer (32 -bit) ，相容性會比較好。

下載下來後，請打開該安裝檔。安裝的過程需要系統管理員帳號，請先準備好。接著點藍色的文字連結打開安裝檔，你應該會看到這樣的介面：

請勾選底下的 Add python.exe to PATH，對未來的設定會比較方便。

再來直接點最上面的 Install Now ，等一下

等到這樣，已經很完美了！這代表你的Python應該準備好了，也就是我們可以開始下一步。按下Close關掉這個頁面，因為我們不會需要這個檔案了。

檢查 Python ​

在那之前，做完都先驗證是否成功是一件很重要的事情！因此，請讓我們按下 Windows+R 鍵，輸入 cmd 之後按下Enter鍵，並在接下來跳出的黑底視窗中輸入以下文字：

bash
$ python --version

如果你出現類似這樣的畫面，代表一切都很完美：

bash
$ python --version
Python 3.10.6

接下來我們就可以開始安裝機器人本體了！請進入下一頁，讓我們開始吧~~
