這是一個很棒的Discord機器人，快點跟我們一起來了解它吧！

為什麼選擇雲羽機器人？ ​

雲羽機器人擁有以下幾種特色，它們方便了你的開發：

　開放原始碼

沒錯，這個機器人是開源的。

這代表你可以任意的修改機器人的原始碼，擴充、簡化、翻譯、配置...

更棒的是，如果你很喜歡自己的作品，你甚至可以將它分享給我們，然後...你的大名甚至會出現在我們的網站上！

　基於 Python 開發

這個機器人使用著名的 Python 程式語言設計、開發。

這代表著它有各式各樣的可能性：你將可以讓它監控你的 Minecraft 伺服器、讓它向你伺服器中的人介紹、甚至可以串 Gemini API，讓你的機器人真的有了自己的意識

因為 Python 的著名，你可以讓它與任何東西整合，讓它變成服務體系的一部份！

　簡單的配置文件

本機器人的配置文件語法採用簡單易懂的 YAML 標記語言。

因為如此，它讓你不需要動任何主程式的部分；你只需要想辦法盡力擴充機器人的能力，然後向大家證明你的群組。它就像一個堅強的夥伴，在你背後默默支持著你！

正因如此，我們可以很敢說，你一定沒看過這麼簡單的程式碼！
