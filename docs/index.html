說明文件
前往官網

簡單的操作與維護

本機器人採用大受好評的 Python 與 Discord.Py，讓你可以不必擔心程式碼的錯誤。不只如此，這些程式碼還比基岩穩定！

豐富的擴展與外掛

只要你會使用 Cog 架構下的 Discord.py，想要什麼功能就能擁有。你說你不會？預設就有很多可以用了啊！

開源的維護與支持

這個機器人的原始碼全開放在 Github 中，你可以任意使用它建立屬於自己的全新機器人。只要記得，記得要繼續開源、幫助大家！

快速的操作與指令

機器人盡力使用 Discord 提供的介面組件，讓你不用跟指令打交道、不用背指令語法、更不用看到複雜的東西！
